Antivirus - Malware Detection with SHA-256

This project involved developing an antivirus application that detects malware using SHA-256 hashing. The project was developed in C#.

Introduction

The aim of this project was to create an efficient antivirus application that could scan files for malware by comparing their SHA-256 hash values against known malicious hashes. This approach ensures that even slight changes in file content are detected as potential threats.

Screenshot of the antivirus application interface, where users can initiate a scan and view results.

Technical Implementation

The antivirus software was developed in C#, utilizing the SHA-256 cryptographic hash function to generate unique identifiers for files. The application compares these hashes with a database of known malware signatures to identify threats.

Diagram illustrating the SHA-256 hashing process, which converts file data into a fixed-size string of characters, unique to the file content.

Challenges and Solutions

One of the main challenges was ensuring the application could process large files quickly while maintaining accuracy. By optimizing the hashing and comparison algorithms, the application can efficiently scan files without compromising performance.

Performance metrics of the antivirus software, showing scan times for various file sizes and types.

Conclusion

This antivirus project demonstrated the effectiveness of SHA-256 in malware detection. The use of C# provided the necessary tools to develop a responsive and reliable application, capable of protecting systems from known threats.
